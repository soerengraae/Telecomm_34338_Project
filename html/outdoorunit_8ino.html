Function Documentation

◆ collectRealWeather()

void collectRealWeather ( )

Reads temperature, humidity, and wind speed values, storing them in global variables.

  • Temperature is stored in temp (°C).
  • Humidity is stored in humidity (%).
  • Wind speed is stored in windSpeed, based on an analog read from windPin.
Note
The current implementation only stores the raw analog reading as the wind speed. A user-defined formula should be applied to convert this to a real speed in m/s.

◆ fetchRainStatus()

int8_t fetchRainStatus ( )

Fetches and interprets the weather forecast to determine the rain status.

Uses the OpenWeatherMap API forecast data for the next two intervals (3-hour blocks). Checks if "Rain" or "Drizzle" is mentioned in the first or second forecast blocks, and assigns a status code accordingly.

Returns
An integer representing the success or failure of the operation.
  • 0,1,2,3: Valid status for expected rain (see rainStatus).
  • -1: WiFi not connected.
  • -2: HTTP GET request failed.
  • -3: JSON deserialization failed.

◆ loop()

void loop ( )

Arduino main loop function. Reads sensor data, retrieves forecast, sends data to ThingSpeak, and waits 10 minutes.

The cycle is:

  1. Delay 3 seconds to ensure the DHT sensor refreshes data.
  2. Collect real-world weather data using collectRealWeather().
  3. Retrieve rain status from fetchRainStatus().
  4. Initialize ThingSpeak, connect, update fields, and disconnect.
  5. Delay 10 minutes before the next cycle.

◆ setup()

void setup ( )

Arduino setup function. Initializes serial communication, DHT sensor, and connects to WiFi.

◆ rainStatus

int8_t rainStatus = 0

Stores the computed rain status based on the forecast data.

The mapping used is:

  • 0: No rain expected in the next two intervals (6 hours).
  • 1: Rain expected in the first interval (first 3 hours).
  • 2: Rain expected in the second interval (next 3 hours).
  • 3: Rain expected in both intervals (first and second).
  • Negative values returned by fetchRainStatus() indicate errors connecting or parsing data.
