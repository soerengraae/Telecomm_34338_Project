\doxysubsection{Detailed Description}
Handles Bluetooth communication between ESP32 devices, receiving JSON commands to control GPIO pins. 

This program sets up the ESP32 as a slave device, waiting for a master ESP32 device to connect via Bluetooth. Once connected, it receives JSON data to control specific GPIO pins and sends back acknowledgments.
