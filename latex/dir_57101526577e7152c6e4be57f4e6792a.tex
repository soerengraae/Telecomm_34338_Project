\doxysubsection{Detailed Description}


 

Georgy “\+The Voronator” Voronoy

This is a fast library for computing the \href{https://en.wikipedia.org/wiki/Voronoi_diagram}{\texttt{ Voronoi diagram}} of a set of two-\/dimensional points. It is based on \href{https://github.com/mapbox/delaunator}{\texttt{ Delaunator}}, a fast library for computing the \href{https://en.wikipedia.org/wiki/Delaunay_triangulation}{\texttt{ Delaunay triangulation}} using \href{https://github.com/mapbox/delaunator/blob/master/README.md\#papers}{\texttt{ sweep algorithms}}. The Voronoi diagram is constructed by connecting the circumcenters of adjacent triangles in the Delaunay triangulation.

For an interactive explanation of how this library works, see \href{https://observablehq.com/@mbostock/the-delaunays-dual}{\texttt{ The Delaunay’s Dual}}.\hypertarget{README.md_autotoc_md3245}{}\doxysubsection{\texorpdfstring{Installing}{Installing}}\label{README.md_autotoc_md3245}

Returns the Delaunay triangulation for the given array or iterable of {\itshape points}. If {\itshape fx} and {\itshape fy} are not specified, then {\itshape points} is assumed to be an array of two-\/element arrays of numbers\+: \mbox{[}\mbox{[}{\itshape x0}, {\itshape y0}\mbox{]}, \mbox{[}{\itshape x1}, {\itshape y1}\mbox{]}, …\mbox{]}. If {\itshape that} is specified, the functions {\itshape fx} and {\itshape fy} are invoked with {\itshape that} as {\itshape this}. \# {\itshape delaunay}.{\bfseries{halfedges}}

For each index 0 ≤ {\itshape i} \texorpdfstring{$<$}{<} {\itshape halfedges}.length, there is a halfedge from triangle vertex {\itshape j} = {\itshape halfedges}\mbox{[}{\itshape i}\mbox{]} to triangle vertex {\itshape i}. \Hypertarget{README.md_delaunay_inedges}%
\# {\itshape delaunay}.{\bfseries{inedges}}

The incoming halfedge indexes as a Int32\+Array \mbox{[}{\itshape e0}, {\itshape e1}, {\itshape e2}, …\mbox{]}. For each point {\itshape i}, {\itshape inedges}\mbox{[}{\itshape i}\mbox{]} is the halfedge index {\itshape e} of an incoming halfedge. For coincident points, the halfedge index is -\/1; for points on the convex hull, the incoming halfedge is on the convex hull; for other points, the choice of incoming halfedge is arbitrary. The {\itshape inedges} table can be used to traverse the Delaunay triangulation; see also \doxylink{README.md_delaunay_neighbors}{\texorpdfstring{$\ast$}{*}delaunay\texorpdfstring{$\ast$}{*}.neighbors}.
