A DOM Node that the sheet will insert all of it\textquotesingle{}s style tags into, this is useful for inserting styles into iframes.\hypertarget{README.md_autotoc_md169}{}\doxysubsubsubsection{\texorpdfstring{key}{key}}\label{README.md_autotoc_md169}
This will be set as the value of the {\ttfamily data-\/emotion} attribute on the style tags that get inserted. This is useful to identify different sheets.\hypertarget{README.md_autotoc_md170}{}\doxysubsubsubsection{\texorpdfstring{speedy}{speedy}}\label{README.md_autotoc_md170}
This defines how rules are inserted. If it is true, rules will be inserted with \href{https://developer.mozilla.org/en-US/docs/Web/API/CSSStyleSheet/insertRule}{\texttt{ {\ttfamily insert\+Rule}}} which is very fast but doesn\textquotesingle{}t allow rules to be edited in Dev\+Tools. If it is false, rules will be inserted by appending text nodes to style elements which is much slower than insert\+Rule but allows rules to be edited in Dev\+Tools. By default, speedy is enabled in production and disabled in development.\hypertarget{README.md_autotoc_md171}{}\doxysubsubsubsection{\texorpdfstring{prepend}{prepend}}\label{README.md_autotoc_md171}
{\bfseries{Deprecated\+:}} Please use {\ttfamily insertion\+Point} option instead.
