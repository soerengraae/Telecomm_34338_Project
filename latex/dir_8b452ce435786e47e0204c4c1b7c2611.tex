\doxysubsection{Detailed Description}
Fast robust predicates for computational geometry in Java\+Script. Provides reliable 2D and 3D point orientation tests ({\ttfamily orient2d}, {\ttfamily orient3d}, {\ttfamily incircle}, {\ttfamily insphere}) that are not susceptible to floating point errors (without sacrificing performance). A modern port of \href{https://www.cs.cmu.edu/~quake/robust.html}{\texttt{ Jonathan R Shewchuk\textquotesingle{}s C code}}, an industry standard since 1996.

\begin{DoxyItemize}
\item Returns a {\itshape positive} value if the points {\ttfamily a}, {\ttfamily b}, and {\ttfamily c} occur in {\itshape counterclockwise} order ({\ttfamily c} lies to the left of the directed line defined by points {\ttfamily a} and {\ttfamily b}).
\item Returns a {\itshape negative} value if they occur in {\itshape clockwise} order ({\ttfamily c} lies to the right of the directed line {\ttfamily ab}).
\item Returns {\itshape zero} if they are {\itshape collinear}.
\end{DoxyItemize}

The result is also an approximation of twice the signed area of the triangle defined by the three points.\hypertarget{README.md_autotoc_md3704}{}\doxysubsubsection{\texorpdfstring{{\ttfamily incircle(ax,ay, bx,by, cx,cy, dx,dy)}}{{\ttfamily incircle(ax,ay, bx,by, cx,cy, dx,dy)}}}\label{README.md_autotoc_md3704}

\begin{DoxyItemize}
\item Returns a {\itshape positive} value if the point {\ttfamily d} lies {\itshape outside} the circle passing through {\ttfamily a}, {\ttfamily b}, and {\ttfamily c}.
\item Returns a {\itshape negative} value if it lies {\itshape inside}.
\item Returns {\itshape zero} if the four points are {\itshape cocircular}.
\end{DoxyItemize}

The points {\ttfamily a}, {\ttfamily b}, and {\ttfamily c} must be in {\itshape counterclockwise} order, or the sign of the result will be reversed.\hypertarget{README.md_autotoc_md3705}{}\doxysubsubsection{\texorpdfstring{{\ttfamily orient3d(ax,ay,az, bx,by,bz, cx,cy,cz, dx,dy,dz)}}{{\ttfamily orient3d(ax,ay,az, bx,by,bz, cx,cy,cz, dx,dy,dz)}}}\label{README.md_autotoc_md3705}

\begin{DoxyItemize}
\item Returns a {\itshape positive} value if the point {\ttfamily d} lies {\itshape above} the plane passing through {\ttfamily a}, {\ttfamily b}, and {\ttfamily c}, meaning that {\ttfamily a}, {\ttfamily b}, and {\ttfamily c} appear in counterclockwise order when viewed from {\ttfamily d}.
\item Returns a {\itshape negative} value if {\ttfamily d} lies {\itshape below} the plane.
\item Returns {\itshape zero} if the points are {\itshape coplanar}.
\end{DoxyItemize}

The result is also an approximation of six times the signed volume of the tetrahedron defined by the four points.\hypertarget{README.md_autotoc_md3706}{}\doxysubsubsection{\texorpdfstring{{\ttfamily insphere(ax,ay,az, bx,by,bz, cx,cy,cz, dx,dy,dz, ex,ey,ez)}}{{\ttfamily insphere(ax,ay,az, bx,by,bz, cx,cy,cz, dx,dy,dz, ex,ey,ez)}}}\label{README.md_autotoc_md3706}

\begin{DoxyItemize}
\item Returns a {\itshape positive} value if the point {\ttfamily e} lies {\itshape outside} the sphere passing through {\ttfamily a}, {\ttfamily b}, {\ttfamily c}, and {\ttfamily d}.
\item Returns a {\itshape negative} value if it lies {\itshape inside}.
\item Returns {\itshape zero} if the five points are {\itshape cospherical}.
\end{DoxyItemize}
